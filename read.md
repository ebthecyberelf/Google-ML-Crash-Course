# Google ML Crash Course

This repository contains exercises and materials from the [Google Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course).

## Contents

- **ml/cc/exercises/** - Jupyter notebooks with hands-on ML exercises
  - `linear_regression_taxi.ipynb` - Linear regression exercise using taxi data

## Getting Started

1. Clone this repository
2. Open the notebooks in [Google Colab](https://colab.research.google.com/) or a local Jupyter environment
3. Follow the exercises to learn machine learning concepts
